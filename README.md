# 🌿 Espacio de Calma

Aplicación web para manejar la ansiedad mediante juegos y técnicas de respiración.

## Tecnologías utilizadas
- HTML5 Semántico
- SASS/SCSS (Mobile First)
- JavaScript ES6+ Modules
- PWA (Service Worker)
- Web Speech API
- Canvas API

## Estructura del proyecto
Ver la estructura en la documentación.

## Instalación
1. Clonar el repositorio
2. Abrir con VS Code
3. Instalar extensión Live Sass Compiler
4. Ejecutar Live Server
